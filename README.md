*The Problem statement: 
 A python function that performs the following tasks using My full name: 
 1. count and display the number of vowels and consonants in the name.
 2. Display the ASCII value of each character.
 3. Create a reversed version of the name.
 4. Check if the name is a palindrome (same forward and backward).
 5. Store each unique letter of the name in a Python List and sort it alphabetically.
 6. Display the first non-repeating character in the name (if any).

*Technologies used:
  IDE: Spyder 6
  Language: python

*How to run the code:
  Before running this program, you need to have "Spyder 6" IDE. 
  After opening the program file, you have to run the program to get the output.

*Sample Inputs and Outputs:
  Full Name: Faria Khanam Jerin
  Vowels: 7 ,  Consonants: 9
  ASCII Values:  [70, 97, 114, 105, 97, 32, 75, 104, 97, 110, 97, 109, 32, 74, 101, 114, 105, 110]
  Reversed Name:  nireJ manahK airaF
  Is Palindrome:  False
  Unique Letters (sorted): ['F', 'J', 'K', 'a', 'e', 'h', 'i', 'm', 'n', 'r']
  First Non-Repeating Character:  F
